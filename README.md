# enBox
> A minimal desktop environment suite for Debian.

Brings together Openbox, tint2, picom and conky along with various custom utilities/scripts and other useful tools (many from XFCE) to create a consistent, cohesive and easy to use environment.
Focus is usability, and most important ensuring everything looks the same across the myriad of graphical toolkits and platforms available on Linux.
